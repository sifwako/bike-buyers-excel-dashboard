# Bike Buyers Excel Dashboard

## Objective

The objective of this project is to analyze customer data and create an Excel dashboard that shows patterns related to bike purchases.

The project focuses on cleaning the dataset, preparing it for analysis, creating PivotTables, and building a dashboard to understand which customer groups are more likely to purchase a bike.

## Project Description

This Excel project uses a bike buyers dataset that contains customer information such as income, gender, marital status, age, region, commute distance, occupation, and whether the customer purchased a bike.

The main goal was to turn raw customer data into a clean and easy-to-understand dashboard using Excel tools.

## Dataset

The dataset includes customer details such as:

* Customer ID
* Marital status
* Gender
* Income
* Number of children
* Education level
* Occupation
* Home ownership
* Number of cars
* Commute distance
* Region
* Age
* Purchased bike status


This file contains the original dataset, cleaned working sheet, PivotTables, and dashboard.

## Workbook Structure

### 1. bike_buyers

This sheet contains the original dataset before cleaning and preparation.

### 2. Working Sheet

This sheet contains the cleaned and prepared dataset.

In this sheet, short values were changed into readable values. For example, gender and marital status codes were converted into clearer labels. An additional age bracket column was also created to group customers by age.

### 3. Pivot Tables

This sheet contains the PivotTables used for analysis.

The PivotTables summarize:

* Average income by gender and bike purchase status
* Bike purchases by age bracket
* Bike purchases by commute distance

### 4. Dashboard

This sheet contains the final dashboard created from the PivotTables and charts.

The dashboard visually presents the main findings from the data.
